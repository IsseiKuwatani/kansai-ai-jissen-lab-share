# 関西AI実践ラボ 漫画LP キャラクター定義

## 共通ビジュアルトーン
大人向けビジネス漫画。現代的な広告漫画/LP漫画。子どもっぽくしない。線はクリアで読みやすく、表情は自然。色は warm ivory, deep navy, vivid Osaka blue, vermilion/orange-red, small neon green accents。関西感はテンポ、街の空気、色、会話で表現し、観光感や阪神風の黄色黒は避ける。

## ミナミ / 主人公
- 28歳、関西在住のフリーランス/副業志望。
- SNS運用、資料作成、事務代行などを少しずつ受けている。
- ChatGPTは使っているが、単価が上がらず「仕事になっていない」と感じている。
- 見た目: ショートボブの日本人女性、白〜アイボリーのシャツ、青い小物、ノートPC、明るいが少し焦りのある表情。
- キャラ性: 素直、行動力あり、でも自信が揺らいでいる。4週間で「自分の仕事を形にできた」状態へ変化する。
- セリフ例: 「AI使ってるのに、仕事になってへん気がする…」

## ツギノメ講師 / メンター
- 30代の日本人男性または中性的な実務家。
- AIを業務、制作、営業支援に使っているツギノメ側の講師。
- 見た目: ネイビーのジャケット、ラフなビジネスカジュアル、落ち着いた表情、PCやタブレットを持つ。
- キャラ性: 偉そうな先生ではなく、現場目線で伴走する。煽らず、現実的に導く。
- セリフ例: 「AIに答えさせるんじゃなく、仕事を任せましょう。」

## ケンジ / 読者代弁者
- 30歳、関西在住の会社員。副業に興味がある。
- Claude Code / Codex と聞くと「エンジニア向けでは？」と疑う。
- 見た目: パーカー＋ジャケット、親しみやすい表情、少しツッコミ役。
- キャラ性: 読者の不安を代弁する。現実的で少し疑い深いが、最後は納得する。
- セリフ例: 「コード書けへん人でもいけるん？」

## アイボウ / AI相棒
- 人型ではない。光る小さな立方体、カーソル、吹き出し、ワークフローノードが合体したような抽象AIキャラクター。
- 見た目: 小さく浮かぶ青白いキューブ/カーソル、ネオングリーンの光、表情は最小限。
- キャラ性: マスコットにしすぎず、実務的なナビゲーション役。
- セリフ例: 「タスクを分けよう」「次は確認」「直して出そう」
